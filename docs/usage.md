# Usage Guide

## Step 1: Run the program

sudo ./deauth

## Step 2: Select interface
Choose a wireless interface (e.g., wlan0)

## Step 3: Monitor mode
The tool automatically enables monitor mode

## Step 4: Enter AP MAC
You can:
- Enter manually
- Type SCAN to auto-discover

## Step 5: Configure
- Packets per second (PPS)
- Duration
- Broadcast mode

## Step 6: Execution
The tool starts sending frames and displays stats.

## Step 7: Report
After stopping, a report file is generated.
